# システム共通仕様書

## フォームバリデーション仕様

### 概要
フォーム保存時に未入力項目があった場合の挙動を統一する。

### 仕様

#### 未入力チェック時の挙動
保存ボタン押下時に必須項目が未入力の場合：

1. **アラート表示**: 未入力項目を示すメッセージをalertで表示
2. **フォーカス移動**: 該当フィールドにカーソルを移動
3. **スクロール**: 該当フィールドが画面中央に表示されるようスクロール
4. **折りたたみ要素の展開**（該当する場合）: アコーディオン、タブ、モーダル等の中に項目がある場合は先に展開する

### 実装例

#### 基本パターン

```javascript
// フォーカス移動用関数
const focusAndScroll = function(element) {
    if (element) {
        element.focus();
        element.scrollIntoView({ behavior: 'smooth', block: 'center' });
    }
};

// バリデーションチェック例
if (!fieldElement.value.trim()) {
    alert('○○を入力してください。');
    focusAndScroll(fieldElement);
    return false;
}
```

#### 折りたたみ要素（アコーディオン等）内の項目の場合

```javascript
const openCollapseAndFocus = function(element, collapseId) {
    const collapseElement = document.getElementById(collapseId);
    if (collapseElement && !collapseElement.classList.contains('show')) {
        new bootstrap.Collapse(collapseElement, { show: true });
        // 展開アニメーション完了後にフォーカス（400ms待機）
        setTimeout(function() {
            element.focus();
            element.scrollIntoView({ behavior: 'smooth', block: 'center' });
        }, 400);
    } else {
        element.focus();
        element.scrollIntoView({ behavior: 'smooth', block: 'center' });
    }
};

// 使用例
if (!fieldElement.value.trim()) {
    alert('○○を入力してください。');
    openCollapseAndFocus(fieldElement, 'detailCollapseId');
    return false;
}
```

#### タブ内の項目の場合

```javascript
const openTabAndFocus = function(element, tabId) {
    const tabTrigger = document.querySelector(`[data-bs-target="#${tabId}"]`);
    if (tabTrigger) {
        const tab = new bootstrap.Tab(tabTrigger);
        tab.show();
        setTimeout(function() {
            element.focus();
            element.scrollIntoView({ behavior: 'smooth', block: 'center' });
        }, 200);
    }
};
```

## Bootstrapバリデーションスタイル無効化

### 概要
`django_bootstrap5`の`bootstrap_field`を使用したフォームで、保存時に枠の色が変わりアイコンが表示される仕様を無効化する。

### 無効化用CSS

```css
/* Bootstrapのバリデーションスタイルを無効化 */
.form-control.is-valid,
.form-control.is-invalid,
.form-select.is-valid,
.form-select.is-invalid {
    border-color: #ced4da !important;
    background-image: none !important;
    padding-right: 0.75rem !important;
}

.form-control.is-valid:focus,
.form-control.is-invalid:focus,
.form-select.is-valid:focus,
.form-select.is-invalid:focus {
    border-color: #86b7fe !important;
    box-shadow: 0 0 0 0.25rem rgba(13, 110, 253, 0.25) !important;
}

.valid-feedback,
.invalid-feedback {
    display: none !important;
}
```

### 適用方法
該当ページの`<style>`タグ内に上記CSSを追加する。
