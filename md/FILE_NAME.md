# Assessments Templates ファイル一覧

## ファイル名と説明

### assessment_list.html
アセスメント一覧表示ページ。
- 全アセスメントをテーブル形式で表示
- 利用者名、実施日、種別、作成者、ステータス（完了/下書き）を表示
- 詳細ボタン、下書きの場合は編集ボタンも表示
- アセスメントがない場合は作成を促すメッセージを表示
- URL: `/assessments/`

### assessment_detail.html
アセスメント詳細表示ページ。
- 基本情報：利用者名、実施日、作成者、種別、面談場所、ステータス
- 利用者情報：ID、年齢、性別、介護度
- アセスメント内容：現在は「開発中」メッセージのみ表示
- 管理情報：作成日時、最終更新日時
- 下書き状態の場合は「編集」ボタンを表示
- アセスメント一覧から詳細ボタンでアクセス

### detailed_assessment_form.html
詳細なアセスメント作成・編集フォーム（共通テンプレート）。
- `is_edit` フラグで新規作成と編集を切り替え
- タブ形式で複数のセクションに分かれている
- 保険情報、健康状態、ADL、IADLなど多数の項目を含む
- ファイルサイズ：278KB（非常に大規模）
- URL: `/assessments/create/` (新規作成)

### assessment_delete.html
アセスメント削除確認ページ。
- 削除前に利用者情報とアセスメント情報を表示
- 取り消し不可の警告表示
- 削除ボタンクリック時にJavaScriptの確認ダイアログを表示
- キャンセルボタンで一覧に戻る

### assessment_print.html
アセスメントシート印刷用ページ（簡易版）。
- A4横向きのExcel様式
- 基本情報、保険情報、家族情報、健康状態、身体状況、ADL、IADL、総合所見・支援目標、作成者情報を表示
- 画面右上に「印刷する」ボタン、左上に「戻る」ボタン
- 現在URLルーティング未設定（実装途中）

### assessment_print_exact.html
アセスメントシート印刷用ページ（完全版）。
- A4縦向き2ページ構成
- 実際のExcelシートの列幅・行高を完全再現（37列）
- 1ページ目：基本事項、保険情報、家族状況、住居状況、健康状態、利用サービス、本人・家族の希望、生活歴、特記・備考
- 2ページ目：身体状況（詳細）、基本動作、ADL、IADL、認知、各項目の課題分析欄
- 現在URLルーティング未設定（実装途中）

## 削除済みファイル
- `assessment_create_simple.html` - 簡易アセスメント作成ページ（不要のため削除）
- `assessment_edit.html` - 編集ページ（開発中メッセージのみ、不要のため削除）
- `detailed_assessment_form_old.html` - 旧バージョンのフォーム（不要のため削除）
