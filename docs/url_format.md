# URL一覧の出力フォーマット

## 指示
ユーザーが「今あるHTMLのURLをすべて教えて」と伝えた場合、以下のフォーマットで出力すること。

## フォーマット例

```markdown
## カテゴリ名
- **機能名**: `http://127.0.0.1:8000/具体的なパス/`
  - テンプレート: `templates/アプリ名/テンプレート名.html`

- **機能名（動的パラメータあり）**: `http://127.0.0.1:8000/パス/<パラメータ説明>/`
  - テンプレート: `templates/アプリ名/テンプレート名.html`
```

## ルール
1. カテゴリごとにセクション分け（## 見出し）
2. 各URLを**太字**で表示
3. URLは具体的に記載（動的パラメータは`<説明>`形式で記載）
4. 対応するテンプレートファイルのパスを明記
5. 関連するURLはグループ化して表示
6. バックアップファイルや作業用ファイルは「その他のテンプレート」セクションにまとめる
